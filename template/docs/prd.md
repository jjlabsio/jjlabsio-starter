# 2주 MVP PRD 템플릿

> **전제**: 이 PRD를 작성하기 전에 `brand_brief.md`가 완성되어 있어야 한다.
> 브랜드 브리프에서 정의한 네이밍, 용어 사전, 톤앤매너, 랜딩 카피를 이 문서 전체에 일관 적용한다.
>
> **사용법**: "brand_brief.md를 참조해서 이 템플릿으로 2주 MVP PRD를 만들어줘."

---

## 1. 제품 개요

### 1.1 한 줄 요약

| 항목           | 내용                                   |
| -------------- | -------------------------------------- |
| **서비스명**   | → `brand_brief.md` 3.1 최종 선정       |
| **한 줄 설명** | → `brand_brief.md` 1.2 엘리베이터 피치 |
| **타겟**       | → `brand_brief.md` 2. Primary Persona  |

### 1.2 MVP 목표

이 MVP가 2주 뒤에 달성해야 하는 것:

- **검증할 가설**: [예: "1인 셀러는 AI 검색에서 자기 브랜드 노출 현황을 확인하기 위해 월 2만원을 지불할 의향이 있다"]
- **성공 기준**: [예: 랜딩 페이지 전환율 3% 이상 / 웨이트리스트 100명 / 유료 전환 10명]
- **MVP 이후 판단**: 성공 기준 충족 시 [다음 단계], 미충족 시 [피봇/중단 기준]

### 1.3 스코프 정의

| 포함 (In Scope)                       | 제외 (Out of Scope)                 |
| ------------------------------------- | ----------------------------------- |
| [MVP에 반드시 들어가는 것]            | [V2 이후로 미루는 것]               |
| [예: 핵심 기능 1~3, 랜딩, 인증, 결제] | [예: 팀 기능, 고급 분석, 모바일 앱] |
| [예: 이메일 알림]                     | [예: 슬랙 연동, 웹훅]               |
